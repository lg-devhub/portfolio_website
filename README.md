# BentoVibe Portfolio

A modern, high-end bento-style portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Bento Grid Layout**: Strictly locked 100vh viewport for an app-like experience.
- **Multi-language Support**: Seamless switching between Portuguese and English.
- **Dark/Light Mode**: Aesthetic theme toggle with persistent user preference.
- **Interactive UI**: Hover-driven blur effects and smooth scale transitions.
- **Shared Components**: Modular sidebar and translations for easy maintenance.

## Project Structure
- `index.html`: Main landing page (Bento Grid).
- `pages/`: Individual sections (About, Projects, Stack, Contact).
- `scripts/`: Shared logic and translation dictionaries.
- `styles/`: Cleaned and organized CSS design system.
- `img/`: Optimized visuals and brand assets.

## Maintenance
To update the sidebar content (avatar, name, bio, social links), edit the `getSidebarTemplate` function in `scripts/script.js`.
To add or modify translations, edit `scripts/translations.js`.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Inter, Outfit, Playfair Display)
